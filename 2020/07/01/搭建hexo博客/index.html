搭建hexo博客

Posted on 2020-07-01

Hexo简介

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

Hexo搭建步骤

1.安装Git(Windows)

Git官网

我选择了腾讯软件中心搜索Git,或者淘宝镜像，因为访问到Git官网实在太慢了

随意在任意桌面右键,有Git Bash代表安装成功！

Git Bash可以使用Win自带的命令

2. 安装nodejs（Windows）

nodejs官网下载

选择LTS(长期支持)

安装完后，打开命令行

3. 安装hexo

输入命令

确认命令

找个文件夹,初始化hexo
